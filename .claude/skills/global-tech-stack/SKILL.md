# Global Tech Stack

## When to use this skill:

- When choosing between libraries or frameworks for a new feature
- When setting up a new project or initializing dependencies
- When deciding on database technologies (SQL vs NoSQL, PostgreSQL vs MySQL, Redis)
- When selecting frontend frameworks (React, Vue, Svelte, Next.js) or backend frameworks (FastAPI, Express, Django, Rails)
- When configuring build tools, bundlers, or package managers (npm, pnpm, poetry, pip)
- When making infrastructure decisions (Docker, Kubernetes, AWS, GCP, Vercel)
- When choosing state management solutions (Redux, Zustand, Pinia, Context API)
- When selecting authentication strategies (JWT, OAuth, sessions, API keys)
- When integrating third-party services or APIs
- When ensuring new code aligns with existing project technology choices
- When reviewing package.json, requirements.txt, pyproject.toml, or similar dependency files
