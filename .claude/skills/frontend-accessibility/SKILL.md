# Frontend Accessibility

## When to use this skill:

- When creating new UI components or pages
- When working on `.jsx`, `.tsx`, `.vue`, `.svelte`, or HTML template files
- When adding form inputs, buttons, or interactive elements
- When implementing modals, dialogs, dropdowns, or other dynamic UI patterns
- When choosing color schemes and ensuring sufficient contrast ratios (4.5:1 for normal text)
- When adding alt text to images or labels to form fields
- When implementing keyboard navigation and focus management
- When using ARIA attributes for complex components
- When testing with screen readers (VoiceOver, NVDA, JAWS)
- When structuring heading hierarchies (h1-h6) for document outline
- When building single-page applications with dynamic content updates
