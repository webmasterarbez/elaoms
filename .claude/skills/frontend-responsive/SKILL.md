# Frontend Responsive

## When to use this skill:

- When creating layouts that need to work on multiple screen sizes
- When working with CSS media queries or Tailwind responsive prefixes (sm:, md:, lg:, xl:)
- When implementing mobile-first CSS patterns (start with mobile, enhance for larger screens)
- When testing UI across different viewport sizes (mobile, tablet, desktop)
- When sizing touch targets for mobile users (minimum 44x44px)
- When optimizing images and assets for mobile network conditions
- When adjusting typography for readability across breakpoints
- When prioritizing content display on smaller screens
- When using relative units (rem, em, %) instead of fixed pixels
- When creating fluid, percentage-based container widths
- When working on `.css`, `.scss`, or component files with responsive styles
