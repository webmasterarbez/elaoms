# Frontend Components

## When to use this skill:

- When creating new React, Vue, Svelte, or other framework components
- When working on files in `/components/`, `/ui/`, `/shared/`, or `/features/` directories
- When defining component props, interfaces, or type definitions
- When refactoring large components into smaller, focused pieces
- When implementing component composition patterns (children, slots, render props)
- When managing component state and deciding where state should live
- When creating component documentation or Storybook stories
- When reviewing component architecture and API design
- When building design system or UI library components
- When handling component lifecycle and side effects
- When working on `.jsx`, `.tsx`, `.vue`, `.svelte` files
