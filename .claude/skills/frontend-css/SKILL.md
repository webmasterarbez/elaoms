# Frontend CSS

## When to use this skill:

- When styling components with CSS, SCSS, or Tailwind classes
- When working on `.css`, `.scss`, `.less`, `.module.css`, or styled component files
- When using Tailwind CSS utility classes for styling
- When creating or modifying design tokens (colors, spacing, typography)
- When implementing dark mode or theme variations
- When fixing CSS specificity or override issues
- When optimizing CSS for production (purging unused styles, tree-shaking)
- When styling responsive layouts and breakpoints
- When working with CSS-in-JS solutions (styled-components, Emotion)
- When maintaining design system consistency
- When avoiding excessive custom CSS by leveraging framework utilities
